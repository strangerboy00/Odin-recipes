# Odin-recipes

# Simple Recipe Website

A basic, responsive recipe website built entirely with semantic HTML. This project showcases fundamental web development skills, including structural layout, media integration, and content organization.

## 🚀 Features

*   **Semantic Structure**: Built using modern HTML5 elements for accessibility and SEO.
*   **Multi-Page Navigation**: Includes a homepage linking to distinct recipe pages.
*   **Rich Media**: Structured presentation of images, preparation times, and instructions.
*   **Clean Layout**: Organized content using lists, tables, and typography.

## 🛠️ Technologies Used

*   **HTML5**: Structure and content.

## 📂 Project Structure

```Odin-recipes
├── index.html                    # Homepage with the recipe directory
├── recipes/                      # Folder containing individual recipes
│   ├── pizza.html                # Recipe page 1
│   ├── burger.html               # Recipe page 2
│   └── blackforestcake.html      # Recipe page 3
├── images                       # Images and media files
└── README.md                     # Project documentation
```

## 📖 What I Practiced

*   Using tags like `<head>, <body>, <title> ,<meta>`.
*   Implementing absolute and relative hyperlinks (`<a>`).
*   Organizing data with ordered (`<ol>`) and unordered (`<ul>`) lists.
*   Displaying structured data using HTML tables (`<table>`).
*   Embedding and optimizing web images (`<img>`).

## 💻 How to Run This Project

1.  Clone this repository to your local machine:
    ```bash
    git clone git@github.com:strangerboy00/Odin-recipes.git
    ```
2.  Navigate into the project directory:
    ```bash
    cd Odin-recipes
    ```
3.  Open `https://strangerboy00.github.io/Odin-recipes/` in any web browser to view the site.
